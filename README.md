# Fake API README

Fake API is an example API in Sinatra that shows how to fake an API with random fake data.

## Install & Run

`gem install`
`rackup`

## Generate Random Data

`rake pirates`

Generates 1,000 random names ands writes a JSON file to `./assets/pirates.json`

# Available Endpoints

`/api/register`
`/api/login`
`/api/pirates`
`/api/team`
